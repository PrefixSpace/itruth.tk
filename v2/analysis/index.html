Analysis Project v2 - Custom Analysis

What is Analysis Project v2?

Analysis Project v2 provides a platform to make tests.

How to use it?

You need to make a json file which contains test informations first. Example.json
Then combine "https://itruth.tk/v2/analysis/customAnalysis.html?json=" with your json file's url.
Like this: https://itruth.tk/v2/analysis/customAnalysis.html?json=https://itruth.tk/v2/analysis/example.json

Is it a free service?

Yes. You can use it without any fee.
